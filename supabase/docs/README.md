# Supabase Documentation

โฟลเดอร์นี้เก็บเอกสารและคำแนะนำเกี่ยวกับการใช้งาน Supabase ในโปรเจคนี้

## 📁 ไฟล์ในโฟลเดอร์นี้

### `HOW_TO_APPLY.md`
คำแนะนำวิธีการ apply SQL queries บน Supabase UI
- วิธีเปิด Supabase Dashboard
- วิธีรัน SQL queries
- วิธีตรวจสอบผลลัพธ์

### `APPLY_MIGRATION_LEADS_FIELDS.md`
เอกสารเกี่ยวกับ migration สำหรับเพิ่มความยาว fields ในตาราง Leads
- อธิบายปัญหา
- วิธีแก้ไข
- SQL queries ที่ใช้

## 📚 เอกสารเพิ่มเติม

สำหรับเอกสารอื่นๆ ที่เกี่ยวข้อง:
- **Migrations**: ดูใน `migrations/` สำหรับ database schema changes
- **SQL Scripts**: ดูใน `scripts/` สำหรับ SQL queries ต่างๆ
- **Edge Functions**: ดูใน `functions/` สำหรับ backend functions

---

**หมายเหตุ:** ไฟล์เหล่านี้ถูกย้ายมาจาก root ของ `supabase/` เพื่อให้โครงสร้างโปรเจคเป็นระเบียบมากขึ้น
